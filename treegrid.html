RAIOS - TreeGrid Editable

Esta es una aplicación React. Para ver el TreeGrid en funcionamiento, ejecuta npm install y luego npm run dev en la terminal.

El TreeGrid incluye:

  • Estructura jerárquica con nodos expandibles/colapsables
  • Celdas editables (click para editar)
  • Ordenamiento por columnas
  • Agregar nuevos items
  • Edición de nombres de nodos
